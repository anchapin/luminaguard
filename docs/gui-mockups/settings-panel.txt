Settings Icons:
  ☑ = Checkbox, checked
  ☐ = Checkbox, unchecked
  ◉ = Radio button, selected
  ○ = Radio button, unselected
  ▼ = Dropdown selector
  [ ] = Text input field
  ✓ = Boolean toggle, on

Color Coding:
  Green = Safe/Enabled
  Red = Danger/Disabled
  Gray = Optional setting
  Blue = Link or action

Trusted Peer Status:
  ● Green dot = Verified and trusted
  ○ Yellow dot = Pending verification
  ✗ Red dot = Revoked trust
